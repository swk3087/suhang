# 수행 일정 보드

인증 없이 동작하는 수행 일정 관리 웹서버입니다.

## 주요 기능

- 공개 홈(`/`): 활성 일정 먼저, 비활성 일정 다음 순서로 제목 목록 표시
- 일정 상세(`/schedule/:id`): 본문, 파일 첨부, 링크 첨부, 텍스트 첨부 표시
- 관리자(`/adminpw/`): 인증 없이 생성/수정/삭제/활성화·비활성화/순서 변경
- 첨부:
  - 파일 여러 개 업로드 가능 (선택사항)
  - 링크(라벨+URL) 여러 개 첨부 가능
  - 텍스트 첨부 여러 개 추가 가능
- 다운로드/공유:
  - 파일은 다운로드 전용 엔드포인트로 강제 다운로드
  - 일정/파일 링크 공유 버튼 제공
  - 공유 URL은 현재 도메인 기준 절대 URL로 복사/공유됨 (모바일 지원)

## 실행 방법

```bash
npm install
npm run dev
```

- 개발 서버: `0.0.0.0:3000`
- 관리자 페이지: `http://localhost:3000/adminpw/`

## 검증

```bash
npm run lint
npm run build
```

## 데이터 저장 위치

- 일정 메타/본문: `data/schedules/items/*.json`
- 일정 인덱스: `data/schedules/index.json`
- 업로드 파일: `data/files/<scheduleId>/...`

참고: 일정 삭제 시 메타데이터만 삭제되고 업로드 파일은 디스크에 유지됩니다.
